# Loops

## What are loops ?

- Loops are used to do the repetitive tasks.

- Loop will stop when the condition fails to satisfy.

## Types of loops

1. for loop
2. while loop
3. do while loop

## Advantages of loops

1. Code reusability
2. saves time
3. traversing the data structures.

## Basic syntax of loops

1. loop starts (declaration and initialization of variables)
2. check the condition
3. if the condition is true than the block is executed else loops stops.
4. increment
5. go to step 2.
