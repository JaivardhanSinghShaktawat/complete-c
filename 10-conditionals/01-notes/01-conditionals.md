# Conditionals

## What are conditionals ?

Conditionals are used to make decision.They are used to execute the code based on some condition.

They are also called as control statements.

## Types of conditional statements

1. if statement
2. if else statement
3. if else if else ladder
4. Nested If

## What are flowcharts ?

Flowcharts are graphical representation of any approach/solution/algorithm.
